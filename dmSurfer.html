How I built it

I used HTML and CSS to build and design the page. Then, I used Javascript and the OpenAPI API to write personalized messages based on the input fields.

What problems did I overcome?

I struggled writing the API request and initially used the wrong OpenAI model. After many attempts, I managed to get it working. The next issue I had was passing the data from the input fields to the API request. This was important because I needed to feed the AI data about the person I was messaging and the job description. These are critical components of a cold message.
When launching to the Chrome Store, I committed my project to Github which led to OpenAI disabling my API keys üò≠. This took me down the rabbit hole of ENV, config.js, and hiding API keys.
